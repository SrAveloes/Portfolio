A Trajetória

Trajetória Profissional

Minha jornada na tecnologia começou no lugar mais crítico de qualquer empresa: os dados que guiam as decisões estratégicas. Atuando desde 2018 com BI (Business Intelligence), aprendi a traduzir problemas operacionais complexos — seja em finanças, logística, estoque ou recursos humanos — em visões analíticas limpas e compreensíveis.

Com o tempo, percebi que desenhar gráficos bonitos não bastava se as planilhas de origem estivessem quebradas ou se a coleta dependesse de preenchimento manual lento. A transição para o desenvolvimento de software foi um passo natural: decidi ir à raiz do problema, construindo integrações robustas (APIs) e aplicativos internos eficientes para garantir que os dados nasçam corretos desde o primeiro clique.

Hoje, atuo exatamente nesse cruzamento. Sou a ponte entre a engenharia de dados invisível (ETL, orquestração e banco de dados) e o software que o usuário final adora usar.
